The Fizz Buzz Game

Ah, the classic FizzBuzz

Also a very popular developer coding challenge.

But can you do it without relying on a computer program?

You'll be shown random numbers between 1 and 1000.

If the number is divisible by 3, type fizz.

If the number is divisible by 5, type buzz.

If the number is divisible by 3 AND 5, type fizzbuzz.

Or if niether the above apply, leave it blank
